# New Statutory States Needed
## We Need Exact Statutory Language for These States

The following 7 states have mandatory statutory lien waiver forms. We need the exact statutory text to build compliant templates. Please provide or confirm the language for each.

---

## 1. Arizona (AZ)
- **Statute:** AZ Rev Stat S 33-1008
- **Notarization:** No
- **Key Rule:** Must use mandatory 4 statutory forms. Any modification invalidates the waiver.
- **Waiver Types Needed:** All 4 (conditional progress, unconditional progress, conditional final, unconditional final)
- **NEED FROM COUNSEL:** Exact statutory form text for each of the 4 types, including any required NOTICE language.

## 2. Massachusetts (MA)
- **Statute:** MA Gen L ch 254 S 32
- **Notarization:** No
- **Key Rule:** Only defines a statutory form for "Partial Waiver and Subordination of Lien" used by general contractors. Other waivers are unregulated.
- **Waiver Types Needed:** Determine which types need statutory language vs. generic
- **NEED FROM COUNSEL:** Exact statutory form for the partial waiver. Confirm whether ObraKit (serving subcontractors, not GCs) even needs to implement this form, or if generic waivers suffice for MA subcontractors.

## 3. Michigan (MI)
- **Statute:** MI Comp L S 570.1115
- **Notarization:** No
- **Key Rule:** Pre-work lien waivers are PROHIBITED by statute. All waivers are only effective upon payment of the amount indicated.
- **Waiver Types Needed:** Determine what types MI statute defines
- **NEED FROM COUNSEL:** Exact statutory forms. Also: should ObraKit add a guard preventing generation of "pre-work" waivers for MI projects?

## 4. Mississippi (MS)
- **Statute:** MS Code S 85-7-433
- **Notarization:** YES - REQUIRED
- **Key Rule:** Must use statutory form AND be notarized to be valid.
- **Waiver Types Needed:** Determine from statute
- **NEED FROM COUNSEL:** Exact statutory forms. For notarization requirement: should we add a notary block to the PDF template? Or add a prominent notice that "This waiver is NOT valid until notarized"?

## 5. Missouri (MO)
- **Statute:** MO Rev Stat S 429.016
- **Notarization:** No
- **Key Rule:** Statutory form is ONLY required for RESIDENTIAL projects. Commercial projects can use generic forms.
- **Waiver Types Needed:** Residential statutory forms
- **NEED FROM COUNSEL:** Exact statutory forms for residential projects. Should ObraKit ask users to classify the project type (residential vs. commercial) for MO projects?

## 6. Nevada (NV)
- **Statute:** NV Rev Stat S 108.2457
- **Notarization:** No
- **Key Rule:** Mandatory statutory forms.
- **Waiver Types Needed:** All types defined in statute
- **NEED FROM COUNSEL:** Exact statutory form text for each waiver type.

## 7. Utah (UT)
- **Statute:** UT Code S 38-1a-802
- **Notarization:** No
- **Key Rule:** Mandatory statutory forms.
- **Waiver Types Needed:** All types defined in statute
- **NEED FROM COUNSEL:** Exact statutory form text for each waiver type.
