# Generic Waiver Template
## Used for TX, FL, NY, and Non-Statutory States

This template is used for states that do NOT require specific statutory forms. It follows industry best practices for lien waiver documentation.

**IMPORTANT NOTE:** As of July 1, 2025, Florida (SB 658) now requires statutory forms. We need to create a dedicated Florida template. See file 08-FLORIDA-UPDATE.md.

---

## Template Structure

### Letterhead
```
[COMPANY NAME]                     (14pt bold, navy)
[Company Address]                  (9pt, gray)
```
Separated by a 2px navy bottom border.

### Title (13pt bold, centered, uppercase, navy)
```
CONDITIONAL WAIVER AND RELEASE ON PROGRESS PAYMENT
```
(Changes based on waiver type)

### PROJECT IDENTIFICATION Section
```
Job Location:                      [project address, city, state, zip]
Property Owner:                    [owner name]
```

### PARTIES Section
```
Claimant (Subcontractor):          [company name]
Customer (General Contractor):     [GC name]
```

### PAYMENT DETAILS Section
```
Amount:                            $[amount]
Through Date:                      [date]
Check Maker:                       [name, if provided]
```

### WAIVER AND RELEASE Section

**Conditional Language:**
```
Upon receipt of a check from [check maker or "the customer"] in the sum
of $[amount], payable to [company name], and when the check has been
properly endorsed and has been paid by the bank on which it is drawn,
[company name] waives and releases any mechanic's lien, stop payment
notice, or bond right claims through [through date] for work performed
through the date stated above on the project located at [job location].
```

**Unconditional Progress Language:**
```
The undersigned, [company name], has been paid and has received a progress
payment in the sum of $[amount] for labor, services, equipment, or
materials furnished to the project located at [job location]. The
undersigned hereby unconditionally waives and releases any and all
mechanic's lien, stop payment notice, or bond right claims through
[through date].
```

**Unconditional Final Language:**
```
The undersigned, [company name], has been paid and has received final
payment in full for labor, services, equipment, or materials furnished
to the project located at [job location]. The undersigned hereby
unconditionally waives and releases any and all mechanic's lien, stop
payment notice, or bond right claims for all work performed on the project.
```

### WARNING BOX (Only for Unconditional Waivers)
Orange border box with warning text:
```
WARNING: This document waives rights unconditionally and is enforceable
upon signing, even if payment has not been received. Only sign this
document if you have confirmed receipt of payment.
```

## Questions for Counsel

1. Is our generic waiver language legally sufficient for:
   - Texas (Tex. Prop. Code SS 53.281-53.286)
   - New York (N.Y. Lien Law SS 3-39-a)
   - Other non-statutory states

2. Does Texas actually require statutory forms post-HB 2237? Some sources list TX as a "statutory state" but our understanding is that TX eliminated statutory form requirements.

3. Should our generic template include any additional clauses for enforceability (e.g., consideration clause, governing law)?

4. Is the WARNING box on unconditional waivers legally helpful or could it create liability? (i.e., if a user signs despite the warning)

5. Should the "Claimant (Subcontractor)" label be changed to something more legally precise for states where the terminology differs?
