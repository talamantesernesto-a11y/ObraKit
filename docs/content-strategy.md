# ObraKit Content Strategy & Keyword Research

## Executive Summary

ObraKit has a **massive competitive advantage** in organic search: **zero competition in Spanish-language lien waiver content**. While English keywords are dominated by Levelset, Siteline, Procore, and LegalTemplates, nobody serves Hispanic subcontractors in their language.

**Strategy**: Own the Spanish-language lien waiver niche first, then build English authority through state-specific pages and competitor comparison content.

---

## Competitive Landscape

| Competitor | Strength | Weakness (ObraKit's Opportunity) |
|---|---|---|
| Levelset (Procore) | Dominates "lien waiver" head terms, massive blog library | Enterprise pricing, English-only, complex UX |
| Siteline | Strong subcontractor content, modern SaaS | English-only, focused on large commercial subs |
| LegalTemplates | Ranks for "free form" queries | Static PDFs, no state compliance, no Spanish |
| LienWaiver.pro | Quick generation tool | No educational content, no Spanish, no mobile |
| pdfFiller | Ranks for fill-in forms | Generic tool, no construction expertise |

**Nobody serves Spanish-speaking subcontractors.** This is a blue ocean.

- "Mobile lien waiver app" — no competitor owns this angle
- Field workers need phone-based solutions, not desktop SaaS
- Position as "the lien waiver app you use from the job site"

### 3. State-Specific Depth
- 12 states require statutory forms — create definitive pages for each
- Bilingual state pages (EN + ES) double the keyword coverage
- Include actual form requirements, deadlines, notarization rules

### 4. Subcontractor Advocacy
- Position ObraKit as the subcontractor's ally, not the GC's tool
- Most competitors (Procore, GCPay, Oracle) are GC-focused
- Content about payment rights, lien protection = trust building

## Technical SEO Implementation Notes

### URL Structure
```
/blog/que-es-un-lien-waiver          (Spanish pillar)
/blog/lien-waiver-california         (English state page)
/es/blog/lien-waiver-california      (Spanish state page)
/compare/obrakit-vs-levelset         (Comparison)
/lien-waiver-form                    (Product landing - EN)
/es/formulario-lien-waiver           (Product landing - ES)
```

### Schema Markup per Page Type
- **Blog posts**: Article schema with author, datePublished, dateModified
- **State pages**: FAQPage schema (allowed for legal/gov info) + BreadcrumbList
- **Comparison pages**: SoftwareApplication schema for each product
- **Landing pages**: Product schema with offers

### Internal Linking Strategy
- Every blog post links to the relevant product landing page
- State pages link to both English and Spanish versions (hreflang)
- Spanish content links to the Spanish pillar ("¿Qué es un Lien Waiver?")
- Comparison pages link to signup/free trial

### hreflang Implementation
```html
<link rel="alternate" hreflang="en" href="https://obrakit.ai/blog/lien-waiver-california" />
<link rel="alternate" hreflang="es" href="https://obrakit.ai/es/blog/lien-waiver-california" />
```
